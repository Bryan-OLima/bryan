Adivinhador de números

A ideia do programa é identificar o numero em que o usuário está pensando. Com perguntas para um resultado lógico através de funções, retorne ao usuário o número em que ele pensou.

O projeto consta com: 54.4% em JavaScript, 33.8% em CSS e 11.8% em HTML.

O Mentalista

Desenvolvido durante a imersãoDev da Alura, o projeto consiste em: o programa gerar um numero randomico de 0 á 10 e o usuário tentar adivinhar em 3 tentativas qual é o número em que o computador escolheu.

O projeto consta com: 26.7% em JavaScript, 45.7% em CSS e 27.6% em HTML.

Calculadora de média

Este programa consiste em tirar média de alunos de escolas baseado em notas bimestrais.

O projeto consta com: 20.5% em JavaScript, 36.3% em CSS e 43.2% em HTML.
